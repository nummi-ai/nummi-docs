## Integration Setup

### **Gmail Integration (Pro)**
1. Navigate to Email section
2. Click "Connect Gmail"
3. Authorize access permissions
4. Wait for initial sync (1-2 minutes)
5. Start using AI summaries and replies

### **Social Media Accounts (Pro)**
1. Go to Content Studio
2. Click platform connect buttons
3. Authorize posting permissions
4. Test with sample post
5. Begin creating and scheduling content
